feat — новая функциональность

fix — исправление бага

docs — документация (README, комментарии)

style — форматирование (пробелы, ;, prettier), без изменения логики

refactor — переработка кода без новых фич и багфиксов

test — тесты

chore — всё служебное (конфиги, CI, зависимости)

build — сборка, зависимости
